# 🍳 Lector de Recetas en Python

Pequeña aplicación en consola que permite:

- 📖 Leer recetas
- ➕ Crear recetas
- 📂 Crear categorías
- ❌ Eliminar recetas
- 🗑️ Eliminar categorías

## 🛠 Tecnologías usadas
- Python
- pathlib
- shutil

## ▶ Cómo ejecutar

1. Clonar el repositorio
2. Ejecutar:

```
python main.py
```

## 📁 Estructura

Las recetas se guardan dentro de la carpeta `Recetas/`
Cada categoría es una carpeta.
Cada receta es un archivo `.txt`.
