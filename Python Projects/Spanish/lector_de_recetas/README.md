# 🍳 Lector de Recetas en Python

Pequeña aplicación en consola que permite:

- 📖 Leer recetas
- ➕ Crear recetas
- 📂 Crear categorías
- ❌ Eliminar recetas
- 🗑️ Eliminar categorías

## 🛠 Tecnologías usadas
- Python
- Librerías estándar: pathlib, os, shutil,


## ▶ Cómo ejecutar

1. Clonar el repositorio
2. Ejecutar:

```
python main.py
```

## 📁 Estructura

Las recetas se guardan dentro de la carpeta `Recetas/`
Cada categoría es una carpeta.
Cada receta es un archivo `.txt`.

## 🚨 Recomendaciones
1. Si vas a añadir alguna receta, hazlo sin cadenas de texto que incluyan el carácter 'ñ' o tildes.
